Detailed Description

This program sets up the pins and timers required for a "MotorDriver" and two "Encoder" objects and creates the objects.

It then turns the motor to spin from 0% PWM to 100% PWM to -100% PWM to 0% and waits to repeat again. Simultaneously, the encoder prints out values until Ctrl-C is pressed, in which it zeros the encoder.
